## 📖 Sobre o projeto
Este projeto é uma página web desenvolvida em **HTML puro**, com o objetivo de apresentar um **cardápio semanal saudável**.

Ele foi criado como atividade prática para treinar:
- Estrutura básica do HTML
- Uso de tabelas
- Organização semântica
- Inserção de imagens

---

## 🗓️ Funcionalidades
✔️ Exibição de cardápio semanal  
✔️ Organização em tabela  
✔️ Informações do nutricionista  
✔️ Layout simples e direto
